About Mapster

Mapster is a web application that allows users to plot custom events on a map and view and experience these events in a narrative story format. Our long-term vision for Mapster is for users to be able to annovate their own maps with events, order these events, save the maps, and playback or view the events in order.

So far the tool has two examples of these kinds of events.

We built this tool for Information Organization Lab, a graduate class at the School of Information at UC Berkeley in California.
